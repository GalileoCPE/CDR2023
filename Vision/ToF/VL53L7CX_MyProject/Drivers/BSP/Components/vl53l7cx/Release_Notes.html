Purpose

The VL53L7CX Component Driver is a driver intenteded to be used within the STM32Cube ecosystem. This software implements a Component Driver running on STM32. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers.

Here is the list of references to user documents:

  • VL53L7CX: Time-of-Flight 8x8 multizone ranging sensor with wide field of view
  • VL53L7CX-SATEL: Breakout board for VL53L7CX
